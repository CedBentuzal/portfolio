# Portfolio Backend (Cloudinary + Vercel)

Serverless API for signed uploads and listing Cloudinary assets.

## Environment Variables (Vercel)
- CLOUDINARY_CLOUD_NAME
- CLOUDINARY_API_KEY
- CLOUDINARY_API_SECRET
- ADMIN_EMAIL
- ADMIN_PASSWORD

## Endpoints
- POST /api/admin-login
- POST /api/cloudinary-signature
- GET /api/cloudinary-list
